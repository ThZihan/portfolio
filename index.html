Achievements

Intra IUB Techfest (Spring23) Award for best Database Project: Student Performance Management System

Developed Student Performance Monitoring System (SPMS 4.0)" as part of Team Failure DataBees . This project aimed to enhance an outcome-based education framework by analyzing student performance through various metrics like quizzes, exams, and course outcomes. We developed a relational database system that allowed for seamless data input, storage, and retrieval, enabling stakeholders to access statistical reports on student and faculty performance. Additionally, the system featured role-based dashboards, backlog record management, and web application.
